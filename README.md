# credit-risk-modeling

# Credit Risk Modeling - Expected Loss

End-to-end credit risk model built on Lending Club consumer loan data (2007–2014).

## What this project covers
- Data preparation & WoE/IV analysis
- Probability of Default (PD) - Logistic Regression scorecard
- Loss Given Default (LGD) - Two-stage model
- Exposure at Default (EAD)
- Expected Loss = PD × LGD × EAD
- Model Monitoring using Population Stability Index (PSI)

## Dataset
Download from Kaggle: https://www.kaggle.com/wendykan/lending-club-loan-data/version/1

## Tools
Python, pandas, numpy, scikit-learn, matplotlib, seaborn
